##插件目的
请尊称APL…(~喵，又是个烂名字，还A开头想占个插件首位…下次叫AAAA算了)。    
APL的目的是：检查文章中的图片，将外链图片下载回本地，然后替换原先的图片，基本上是RPU的逆过程。所以蛋疼的话可以把APL 和 RPU两个联合着用：下载回来、然后上传过去、然后下载回来……   

##使用方法
1.安装，略。安装后可在后台设置中见“将图片下载到本地”。   
2.进入设置本地数据库信息、保存路径(可以留空)、是否使用Curl检查。   
关于Curl检查，需要主机支持，建议勾选，可以判断原链接是否失效，并记录失效文章和链接。   
3.选择需要下载的文章，下载即可(建议选择一篇测试)。如果点后出现空页面，说明主机不支持Curl，去掉勾选后重试。   

##说明
1.同上篇，发布前已尽量测试，但不保证有未知bug，因为涉及数据库操作，请使用前备份。   
2.原创插件，请尊重本人劳动。   
3.插件需要下载远程图片，可能会大量占用主机资源，用前请三思。   
4.后期版本可能会把APL 和 RPU 柔和在一起，并加入Js。
